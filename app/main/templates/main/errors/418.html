I'm a Teapot

418 - I'm a teapot

The server refuses the attempt to brew coffee with a teapot.

What does this mean?

This is a reference to the Hyper Text Coffee Pot Control Protocol (HTCPCP/1.0) defined in RFC 2324. It's an April Fools' joke from 1998 that became an official HTTP status code.
